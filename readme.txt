== Description ==

This plugin allows you show notice bar on top or bottom your page. You can edit height, background, font color and embed links.



- Responsive Web Design

- Easy to Use





Properties:

- Notice Text

- Height

- Position (Top/Bottom)

- Trasparent

- Background Color

- Font Color



== Installation ==

1. Upload plugin to the /wp-content/plugins/ directory

2. Activate the plugin through the Plugins menu in WordPress
